    Overview

    Build your own dual 10" line array cabinet! Plans available

    The LA210 is a powerful, professional line array speaker system. It can be build as either a passive 2 way or as a powered version. It is a "open" design based on proven and practical line array cabinet design that uses off-the-shelf drivers for simplicity, maintainability and affordability. Simply put - it is a straight forward, highly effective line array system that won't break the bank.
